# react-starter-template
A react template for starter to bootstrap the project

## Quick Start
```bash
git clone https://github.com/ericchanky/react-starter-template
cd react-starter-template
npm install
npm start
```

## What's inside?
### libraries
* react
* redux
* redux-persist
* seamless-immutable

### dev tools
* webpack
* webpack-dev-server
* babel
* styled-jsx
* node-sass
* eslint
